3-Paint using the number (1)

Program the robot to copy the numbers in the column left in the right column.

The robot must read the contents of the box, store it in its memory, go two steps forward, then write the content of its memory in the box.

You may need to use a variable

Program the robot to write in the zero column on each line, the sum of the two numbers on the left.

After storing first number of the row in the variable "Robot memory", you will have to modify it to add the second number.

Program the robot to write instead of zeros to right of each line, the sum of the numbers in the left part of the line.
