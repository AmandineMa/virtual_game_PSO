Pousser les caisses 4

Programmez le robot pour qu'il pousse la caisse sur la case marquée.

Programmez le robot pour qu'il pousse les caisses sur les cases marquées.

Le nombre devant le robot indique la colonne où il y a une caisse à pousser.

Les nombres devant le robot correspondent aux coordonnées de la caisse à pousser :

  • Le premier nombre indique la ligne
  • Le second nombre indique la colonne

Le nombre dans le coin en bas à gauche indique le nombre de caisses présentes sur la grille.
Le nombre en bas de chaque colonne indique la ligne sur laquelle il y a une caisse.

Attention, votre programme doit fonctionner sur les trois tests. Regardez-les avant de programmer !

Vous pouvez avoir besoin d'utiliser une variable.
