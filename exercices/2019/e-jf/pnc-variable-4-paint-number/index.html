Ranger les billes 7

Programmer le robot pour qu'il ramasse chaque bille et la mette dans un trou.

Le robot ne peut transporter qu'une bille à la fois.

Le nombre de billes est indiqué devant le robot. Il faut vous en servir !

Une bille est détectée même si elle est déjà dans un trou.

Regardez les trois tests avant de commencer. Le même programme doit fonctionner sur tous les tests.

Vous pouvez avoir besoin d'utiliser une variable.
